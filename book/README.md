= Differential Equations

A book on Ordinary Differential Equations, by Jason Siefken and Bernardo Galvão-Sousa, with a focus on active learning.

== Compiling

The book is written in [Typst](https://typst.app) and requires the fonds located in `./fonts/` to compile correctly.

To compile with the Typst CLI, run
```sh
typst compile book.typ --font-path=./
typst compile slides.typ --font-path=./
```
